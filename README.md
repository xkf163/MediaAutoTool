# MediaAutoTool

## Project Documentation

MediaAutoTool is a tool for automating the processing and management of media files. This documentation provides an overview of how to use the tool effectively.

### Features
- Automated file organization
- Batch processing capabilities
- Integration with various media formats

### Getting Started

1. **Clone the Repository**  
   Use the following command to clone the repository:  
   ```
   git clone https://github.com/xkf163/MediaAutoTool.git
   ```

2. **Installation**  
   Install the required dependencies using:  
   ```
   pip install -r requirements.txt
   ```

3. **Usage**  
   Run the tool using:  
   ```
   python media_auto_tool.py
   ```
